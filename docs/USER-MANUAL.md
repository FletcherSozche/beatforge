## 2. Ilk Proje: 30 Saniyede Bir Beat

### Adim 1: Sablon Yukle
1. **Tarzlar** butonuna tikla (sag ust)
2. **Drum n Bass - Classic** sec
3. ✅ Pattern yuklendi, BPM 174

> Alternatif: Sol ust menusunden **Menu > Yeni Proje** ile bos basla.

### Adim 2: Calistir
1. **▶ Play** butonuna tikla (veya bosluk tusu)
2. Ses duyuluyor mu? 🎧
3. BPM'i azalt/arttir: + / - butonlari

### Adim 3: Duzenle
- **Hucreye tikla:** Adim ac/kapa
- **Alt + surukle (yukari/asagi):** Velocity ayarla
- **Surukle:** Birden fazla hucreyi isaretle
- **Shift + tikla:** Araliktaki tum hucreleri sec
- **Ctrl + A:** Tum pattern'i sec
- **Ctrl + C / V:** Kopyala / Yapistir
- **Sag tik:** Hucre menusu (toggle, vel, kopyala, temizle)

### Adim 4: Disa Aktar
1. **Rec** butonuna tikla (kirmizi yanar)
2. Beat'i cal
3. **Rec**'e tekrar tikla
4. `.wav` dosyasi bilgisayarina iner
5. Audacity, FL Studio, Ableton vb. ile acabilirsin

## 5. 6 Master Efekt

| Efekt | Parametreler | Ne zaman kullan |
|-------|--------------|-----------------|
| **Reverb** | wet, decay | Pad/vokal icin derinlik |
| **Delay** | wet, feedback, time | Dub/dnb lead'ler |
| **Distortion** | wet, amount | Wobble/reese bass |
| **Filter** | cutoff, type | Sub-bass'i temiz tut |
| **Phaser** | wet, rate, depth | Pad/lead hareketi |
| **Compressor** | ratio, threshold, attack | Master glue |

## 7. 10 Dakikalik Sarki Yapimi

1. Bar seciciyi **128**'e getir
2. Ana 4-bar'i olustur (intro)
3. Bar 5-8'i kopyala (gelistir)
4. Bar 9-32'de drop
5. Bar 33-64'te break/fil
6. Bar 65-128 outro
7. Mixer'da volume ayarla
8. Master efektler: Reverb (pad) + Delay (lead) + Compressor (master)
9. Rec ile kaydet, 10 dakikalik WAV cikar

> **Tavsiye:** Undo (Ctrl+Z) ile istemedigin degisikligi geri al.

---

## 8. Vokal Kayit (3 Track)

1. **Vokal** sekmesine gec
2. Vokal 1, 2 veya 3 sec
3. **Kayit Baslat** butonuna tikla
   - Tarayici mikrofon izni isteyecek, kabul et
4. Sarki soyle
5. **Kayit Durdur**'a tikla
6. Pattern'e git, vokal kanalina adim ekle:
   - Vokal 1 icin 1. bar 1. adim
   - Vokal 2 icin 1. bar 9. adim
   - Vokal 3 icin 2. bar 1. adim
7. Play'de vokaller duyulur

> Mikrofon iznini bir kez vermen yeterli.

## 9. A/B Pattern Karsilastirma

Beat'in iki versiyonu uzerinde calismak icin:

1. Ilk versiyonu yap
2. **A/B** butonuna tikla → "A" kaydedildi
3. B versiyonu icin degisiklik yap
4. **A/B**'ye tekrar tikla → B'ye don
5. **A/B** → A'ya don
6. Begendigini sec, **A/B** ile gecis yap

## 10. Proje Kaydetme

- **Otom. kayit:** 30 saniyede bir otomatik (header'daki saat emoji ile gosterilir)
- **Manuel:** Ctrl+S veya Menu > Kaydet
- **Format:** `.bfp` (duz JSON, editlenebilir)
- **Konum (Web):** Dosya indirilir
- **Konum (Desktop):** Kullanicinin Documents klasoru

### Dosya yapisi ornegi:
```json
{
  "version": 1,
  "name": "Benim Beatim",
  "savedAt": "2026-06-07T10:30:00.000Z",
  "project": {
    "bpm": 174,
    "bars": 4,
    "pattern": {
      "bars": 4,
      "steps": 16,
      "tracks": {
        "kick": { "steps": [{ "on": true, "vel": 0.9 }, ...] },
        ...
      }
    }
  }
}
```

## 12. Sik Sorulan Sorular

**S: Ses cikmiyor!**
C: Tarayici sekmesini tikla (AudioContext tarayici politikasi nedeniyle ilk
kullanici etkilesiminden sonra acilir). Veya hoparlorun sesini ac.

**S: Tarayici desteklemiyor?**
C: Chrome 90+, Edge 90+, Firefox 88+, Safari 14+. iOS Safari'de bazi gecikmeler olabilir.

**S: Uygulama kapandiginda calismam kaybolur mu?**
C: Hayir, 30 saniyede bir otomatik kaydedilir. Menuden "Yeni Proje" secmedikce
oturum geri yuklenir.

**S: Telefonum calisir mi?**
C: Evet, PWA olarak Safari/Chrome'a ekle veya App Store/Google Play'den indir.

**S: Bilgisayarima nasil kurarim?**
C: `start-beatforge.bat` (Windows) veya `start-beatforge.sh` (Mac/Linux)
calistir, ya da `npm install && npm run electron:dev`.

**S: MIDI export?**
C: Yakin gelecekte. Su an WAV (master mix) destekleniyor.
